# 📱 MobileShop - Full Stack MERN E-commerce Platform

A modern, high-performance e-commerce solution built with the MERN stack. This project features a seamless shopping experience for users and a comprehensive dashboard for administrators.

## 🚀 Key Features

* **Full E-commerce Functionality:** Complete user flow from product discovery to secure checkout.
* **Pre-Order System:** Special support for pre-ordering upcoming or out-of-stock products.
* **Order Management:** Robust admin panel to track, update, and manage orders (Pending, Confirmed, Processing, Shipped, Delivered).
* **Automated Email Notifications:** * Instant email alerts sent to users upon order placement.
    * Real-time email updates when the Admin approves delivery or cancels an order.
* **Advanced Admin Dashboard:** Centralized control for managing products, categories, users, and discount coupons.
* **Multi-step Checkout:** Professional UI for shipping information, payment method selection, and order review.

## 🛠️ Tech Stack

* **Frontend:** React.js, Tailwind CSS
* **Backend:** Node.js, Express.js
* **Database:** MongoDB
* **Authentication:** JWT (JSON Web Tokens)
* **Email Service:** Nodemailer
